# 📘 MySQL - Resumos do curso
## 📁 Criação do Banco de Dados
Neste projeto, criamos o banco de dados `livraria` utilizando o comando abaixo:
```sql
CREATE DATABASE livraria;
```

O banco será alimentado com as seguintes tabelas:
- tb_livros
- tb_autor
- tb_editora
